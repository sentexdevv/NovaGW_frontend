# 🏆 Nova Gangwar

## 🛠️ Technik, die mitdenkt

- **Clean CSS:** Das Core-Design liegt in der `style.css`, aber die Kontrolle liegt bei dir. Über die `config.css` änderst du alle Farben in Sekunden.

## 🎨 Anpassung

Um die Farben zu ändern, öffne die Datei `html/css/config.css`. Hier kannst du Variablen wie diese anpassen:

```css
--color-primary: #7e14fd; /* Deine Hauptfarbe */
--hud-faction-name: #ffffff; /* Farbe der Fraktionsnamen */
--hud-bar-fill: linear-gradient(...); /* Farbe der Ladebalken */
```
